Hybrid IoT Intrusion Detection via Federated Learning

Machine Learning & Privacy-Preserving Security | Flower Framework

Developed a decentralized intrusion detection system (IDS) using the Aposemat IoT-23 dataset to classify malicious network traffic while preserving data privacy.

  • Architecture: Implemented a 1D-Convolutional Neural Network (1D-CNN) and Autoencoder for dual-stage detection. The Autoencoder achieved 96% accuracy in identifying anomalies based on reconstruction error.
  • Federated Learning: Orchestrated decentralized training across multiple clients using the Flower framework and the FedAvg algorithm, achieving a global model accuracy of 69-73%.
  • Stress Testing: Simulated adversarial attacks, including Label-Flipping Data Poisoning and TCP Connection Flood DoS, to evaluate the resilience of the central aggregation server.
  • Tools: Python, TensorFlow, Keras, Flower, Scikit-learn, MinMaxScaling.

Enhanced Fake News Detection via Transformer & NLP Fusion

Deep Learning | Natural Language Processing

Conducted an exhaustive study on misinformation detection using the TruthSeeker dataset (134,000+ annotated tweets) to identify linguistic patterns in fake news.

  • Feature Engineering: Compared multiple vectorization techniques including TF-IDF, Word2Vec, and Keras Word Embeddings to map textual context into dense high-dimensional vectors.
  • Model Performance: Evaluated seven supervised models (Random Forest, GBC, KNN, etc.). Achieved a peak accuracy of 99.2% using Word Embeddings and 94.7% using a fine-tuned BERT transformer model.
  • Preprocessing Pipeline: Built a robust cleaning engine using NLTK to handle IP addresses, URLs, stopwords, and stemming, ensuring high-quality input for deep learning layers.
  • Tools: Google Colab, BERT, NLTK, Keras, Matplotlib (ROC/AUC Analysis).

Efficient PUF-Based Authentication for IoT WiFi Networks

Hardware Security & Cryptographic Protocol Design

Designed a novel, lightweight authentication and key exchange protocol using Physically Unclonable Functions (PUFs) to secure resource-constrained IoT devices against physical and network-layer attacks.

  • Tamper-Resistant Identity: Leveraged hardware-level "fingerprints" (PUFs) to generate unique, unclonable digital signatures, eliminating the need for storing sensitive static keys on vulnerable non-volatile memory.
  • Protocol Architecture: Engineered a three-phase security framework: One-time Enrollment (secure CRP storage), Initialization, and Mutual Authentication.
  • Attack Mitigation: Integrated nonce-based transformations and XOR operations to neutralize MAC Spoofing, Replay Attacks, and Rogue Access Point (Evil Twin) threats.
  • Efficiency Optimization: Reduced computational overhead to just 8 PUF operations and 4 Hash calls (8P + 4H) with a minimal storage requirement of 6n bits, making it suitable for low-power e2-micro and edge devices.
  • Cryptanalysis: Performed formal security proofs demonstrating resistance to brute-force attacks by utilizing fresh challenge-response pairs for every session.
