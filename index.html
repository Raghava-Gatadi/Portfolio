Certifications and Courses

1. Google Cybersecurity Certificate

This certification provided me with a comprehensive introduction to both Blue Team and Red Team operations, along with the functioning of a Security Operations Center (SOC). Through this course, I gained in-depth knowledge of key concepts, including:

  • Network Security: Understanding the principles of securing network architectures and TCP/IP protocols.
  • Cloud Security: Exploring best practices to secure cloud environments.
  • Incident Response: Learning response mechanisms to mitigate and manage cyber incidents effectively.
  • Security Frameworks: Studied frameworks such as HIPAA, NIST CSF, FedRAMP, PCI DSS, and NIST 800-39, which are critical in establishing robust cybersecurity standards.
  • Vulnerability Assessment: Explored methods to identify, classify, and prioritize vulnerabilities in systems.
  • Cryptography: Gained insights into asset classification, threat analysis, and encryption techniques.
  • SIEM Tools: Hands-on experience with Splunk, understanding how to monitor, analyze, and respond to security threats using log data.

2. Cisco Networking Essentials/ Bits and Bytes of Computer Networking

This course deepened my understanding of network fundamentals, covering:

  • Network Architecture: Layers of the network model (network, transport, and application layers).
  • Networking Services: DHCP, DNS, and other critical services that keep networks running.
  • Troubleshooting: Techniques to diagnose and resolve common networking issues.

3. Pentesting and Web Application Security Courses

These courses equipped me with the knowledge and skills to assess and secure web applications. Key takeaways include:

  • Mastery of the OWASP Top 10 vulnerabilities, such as SQL injection, XSS, and command injection.
  • Proficiency with tools like Burp Suite, Nmap, Msfconsole, and Msfvenom.
  • Set up and exploited Metasploitable and OWASP virtual machines to practice identifying and exploiting vulnerabilities. These labs included three difficulty levels, allowing me to hone my skills in secure coding and attack methodologies.
