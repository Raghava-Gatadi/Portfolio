NYU Research Internship: gittuf Supply Chain Security Project

By Raghava Gatadi • July 15, 2024

Worked as a Research Intern at NYU's Secure Systems Lab on the gittuf open-source project, focusing on supply chain security for Git repositories. Engineered a Go-based CLI wrapper to intercept Git operations and automate Supply Chain Levels for Software Artifacts (SLSA) compliance.

Project Overview

gittuf is a supply chain security tool that applies the TUF (The Update Framework) model to Git repositories. It enables Git repositories to protect themselves against malicious commits, ensuring only authorized developers can contribute to the codebase.

CLI Wrapper Development

Engineered a Go-based CLI wrapper to intercept Git operations and enforce security policies before allowing commits to proceed. The wrapper validates commit signatures, verifies developer authorization, and ensures compliance with established security protocols.

SLSA Framework Integration

Automated Supply Chain metadata management by synchronizing Release System Log (RSL) entries with TUF metadata. This integration ensures that all Git operations are properly logged and verified according to SLSA security standards.

Cryptographic Enhancements

Hardened cryptographic commit signing using ED25519 keys, providing robust protection against tampering and impersonation. Implemented key rotation mechanisms and improved key management processes.

Research Contributions

Contributed to academic research on supply chain security by analyzing existing threats to Git repositories and proposing novel defense mechanisms. Collaborated with faculty and fellow researchers to publish findings.

Technical Skills Applied

  • Go Programming: Developed secure, efficient CLI tools for Git operations
  • Cryptography: Implemented ED25519 key management and signature verification
  • TUF Framework: Applied The Update Framework concepts to Git security
  • SLSA Compliance: Automated software supply chain security protocols
  • Git Internals: Deep understanding of Git objects, refs, and security models

Key Takeaways:

  • Supply chain security is critical for modern software development workflows
  • Proactive security measures can prevent many common Git repository attacks
  • Integration of frameworks like TUF and SLSA enhances overall security posture
  • Academic research plays a vital role in advancing practical security solutions
